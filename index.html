hello world!!

clone a repo: git clone repo_name

add file: git add "file name"

commit: git commit -m 'message'

push to github: git push

check status: git status

get latest update from github: git pull

when conflict happens: edit the codes by your own and push it

check commits: git log

go back to specific commit/version: git reset --hard "commit-hash"

branching

create new branch: git checkout -b 'style'

delete a branch: git branch -d branchname

switch the branch: git checkout branchname

to combine branches: git merge branchname
